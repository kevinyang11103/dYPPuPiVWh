# 大学城水电管理系统 java1237

## 项目概述

大学城水电管理系统是一款基于Spring Boot和Vue的前后端不分离的信息管理系统。系统旨在为大学城提供水电设备的管理与维护功能，包含管理员和用户两个基本角色，以实现高效的设备管理和使用流程。

## 技术栈

- **开发工具**：IDEA 2020.3、VSCode
- **运行环境**：JDK 1.8、Maven 3.6.0、MySQL 5.7、Node.js 14
- **服务端技术**：Spring Boot、MyBatis-Plus
- **前端技术**：HTML、CSS、Layui、jQuery、Bootstrap、Vue、axios、Element-UI

- 消耗设备管理
- 设备申请管理
- 设备派发管理

- 修理计划管理

## 系统角色

- 管理员：负责整个系统的设备管理和用户管理。
- 用户：负责个人设备的使用和维修申请。
